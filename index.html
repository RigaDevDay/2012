Java Day Riga conference 2012 was held at Islande Hotel Riga on 29 November 2012. This was a major Baltics IT event welcoming more 200 Java programmers from the region. Conference was organised by Latvian Java User Group and Oracle.

15 talks was presented by 14 speakers from Russia, Latvia, Estonia, Poland, Finland, Sweden and Italy.
